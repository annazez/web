# Manifesto

## Core Position

This website reflects a strict engineering stance:

- Architecture over improvisation
- Security and reliability over trend-chasing
- Speed with ethical and sustainable defaults
- Privacy by design, not as a retrofit

## Product Intent

The site exists as:

- A clear identity page
- A multilingual public point of contact
- A stable, low-complexity static platform
- A future base for deeper writing and technical publications

## Non-Goals

- No surveillance analytics
- No ad-tech integrations
- No unnecessary client-side frameworks
